# NestJS REST API Tutorial

## Description
Simple tutorial to create an API to save users and its bookmarks.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# start the docker container
$ npm run db:dev:restart

# start the development server
# use npm run start:dev to watch mode
$ npm run start
```

## End-to-end Test

```bash
$ npm run test:e2e
```
